# CurrencyWallet
CurrencyWallet is a simple finance tracking application.

## Libraries
+ MVVM Architecture
+ Architecture Components (Lifecycle, ViewModel, ViewBinding, Room, LiveData)
+ Coroutines, Flow
+ OkHttp Retrofit & Gson for REST API
+ Hilt
+ CustomView
+ Coil
+ Timber
